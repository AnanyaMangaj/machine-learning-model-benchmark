# Classification Assignment

## Datasets Used
1. Titanic Dataset (from Seaborn)
2. Breast Cancer Dataset (from Scikit-learn)

## Algorithms Used
- Logistic Regression
- KNN
- SVM
- Naive Bayes
- Random Forest
- Decision Tree

## Project Structure
- notebooks/ → Jupyter notebooks
- python_files/ → Python scripts
- models/ → Saved models
- screenshots/ → Output images
- app/ → Streamlit deployment

## How to Run
1. Install libraries:
   pip install -r requirements.txt

2. Open JupyterLab:
   jupyter lab

3. Run notebooks inside notebooks folder
